# ⚽ Penalty Shootout Game

A simple console-based **Penalty Shootout Game** written in C#.  
Test your skills as a penalty taker and try to score goals while the goalkeeper tries to save your shots!

---

## Features

- Choose from **9 shooting directions**:
  - Top Left, Top Center, Top Right
  - Center Left, Center, Center Right
  - Bottom Left, Bottom Center, Bottom Right
- Random goalkeeper dives.
- Dramatic console output with pauses for suspense.
- Instructions and menu system.
- Score tracking for **Goals** and **Saved Shots** (future improvements possible).

---

## How to Play

1. Run the game.
2. From the main menu, select **New Game / Start**.
3. Choose a shooting direction by entering a number (1-9):

| Number | Direction       |
|--------|----------------|
| 1      | Top Left       |
| 2      | Top Center     |
| 3      | Top Right      |
| 4      | Center Left    |
| 5      | Center         |
| 6      | Center Right   |
| 7      | Bottom Left    |
| 8      | Bottom Center  |
| 9      | Bottom Right   |

4. Wait for the goalkeeper to dive.
5. If you choose the same direction as the goalkeeper → **SAVED**! 🧤  
   If you choose a different direction → **GOAL!** ⚽

---

## Instructions Menu

From the main menu, you can select **Instructions** to read the rules and tips for the game.  
- Try to mix up your shots.  
- Predict the goalkeeper’s movement.  
- Score as many goals as you can.
